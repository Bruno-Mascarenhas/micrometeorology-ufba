## Getting Started

You will need install those packages to run all scripts.

### Prerequisites

All scripts were made with Anaconda 5.3

* [Anaconda](https://www.anaconda.com/download/)

Basemap
```
conda install -c anaconda basemap
```

NetCDF4
```
conda install -c anaconda netcdf4 
```
